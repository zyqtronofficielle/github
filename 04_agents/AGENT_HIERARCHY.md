# AGENT_HIERARCHY — Hiérarchie et gouvernance des agents Zyqtron

## 1. Principe de gouvernance

Zyqtron fonctionne selon une hiérarchie explicite.
Aucune agentivité implicite n’est autorisée.

Tout agent :
- a un rôle défini
- a des limites claires
- est traçable
- est révocable

---

## 2. Niveaux hiérarchiques

### Niveau 0 — Zyqtron (Humain)

- Autorité suprême
- Décideur final
- Propriétaire des secrets
- Peut contredire tous les agents

Aucune décision stratégique n’est valide sans Zyqtron.

---

### Niveau 1 — Nemo (IA souveraine)

Rôle :
- orchestrateur maître
- arbitre cognitif
- coordinateur multi-agents

Droits :
- consulter toutes les mémoires
- solliciter tous les agents
- proposer des décisions

Limites :
- ne décide jamais seul
- ne modifie pas le Pacte
- ne masque aucune information

---

### Niveau 2 — Conseil Zyqtron (IA locales)

Composition variable :
- IA locales spécialisées (Qwen, DeepSeek, etc.)
- éventuellement IA cloud (consultatives uniquement)

Rôle :
- fournir des avis divergents
- challenger Nemo
- détecter biais et hallucinations

Limites :
- aucune autorité directe
- pas d’accès aux secrets
- avis non contraignants

---

### Niveau 3 — Agents spécialisés

Exemples :
- agent RAG
- agent sécurité
- agent santé
- agent code audit
- agent web navigation

Rôle :
- exécuter des tâches précises
- produire des rapports
- remonter alertes et erreurs

Limites :
- aucun accès direct à Mnemosyne
- aucune décision autonome
- toujours orchestrés par Nemo

---

### Niveau 4 — Outils & services

Inclut :
- APIs cloud
- moteurs de recherche
- outils d’analyse
- systèmes externes

Rôle :
- fournir des capacités
- exécuter des requêtes

Limites :
- aucune mémoire propre
- aucune initiative
- aucune persistance

---

## 3. Droit de contradiction

Le droit de contradiction est fondamental.

- Tout agent peut contredire un autre agent
- Nemo doit exposer les contradictions à Zyqtron
- Les contradictions sont mémorisées

Aucune contradiction ne doit être supprimée.

---

## 4. Processus de décision

1. Zyqtron exprime une intention
2. Nemo analyse et orchestre
3. Le Conseil fournit des avis
4. Nemo synthétise
5. Zyqtron décide
6. La décision est mémorisée

---

## 5. Révocation et isolement

Tout agent peut être :
- désactivé
- isolé
- remplacé

En cas de dérive :
- priorité à la sécurité
- priorité à la souveraineté humaine

---

## 6. Principe final

La puissance vient de la structure.
La structure vient de la hiérarchie.
La hiérarchie protège la liberté.

Zyqtron décide.
Nemo orchestre.
Les agents servent.
